## NEW Requirements

### Requirement: agent-coc-reporting
The reference agent SHALL demonstrate conversational CoC analytics where an administrator asks questions in plain English and the agent queries the analytics API, synthesizes narrative responses, and handles follow-up questions.

#### Scenario: Admin asks about utilization trends
- **WHEN** an admin asks "What was our bed utilization this week?"
- **THEN** the agent calls `get_utilization` with the past 7 days and daily granularity
- **AND** synthesizes: "Average utilization was {rate}% this week. Peak was {day} at {peak}%. {shelter_name} had the lowest utilization at {low}%."

#### Scenario: Admin asks a follow-up question
- **WHEN** the admin follows up with "Compare that to last week"
- **THEN** the agent calls `get_utilization` for the prior 7-day period
- **AND** presents a comparison: "This week's average ({rate1}%) is {up/down} from last week ({rate2}%). The largest change was at {shelter_name} ({delta}%)."

#### Scenario: Admin asks about unmet demand
- **WHEN** an admin asks "What was our unmet demand on the coldest nights this month?"
- **THEN** the agent calls `get_demand_signals` for the month and `get_weather_status` for temperature context
- **AND** synthesizes: "There were {n} zero-result searches on nights below freezing. The most searched population type was {type}. {n2} searches had no results in the {area} area."

#### Scenario: Admin requests HIC export
- **WHEN** an admin says "Generate the HIC report for this quarter"
- **THEN** the agent calls `export_hic_csv` with the appropriate date
- **AND** returns the CSV file with a summary: "HIC export generated: {n} shelters, {total_beds} total beds across {n_types} population types."

#### Scenario: Admin asks for grant narrative
- **WHEN** an admin says "Draft a paragraph about our bed utilization for the HUD grant application"
- **THEN** the agent uses data from recent `get_utilization` and `get_demand_signals` calls
- **AND** generates a narrative using the recommended language from theory-of-change.md
- **AND** includes the disclaimer: "This paragraph was generated by AI from FABT analytics data. Review by the grant writer is recommended before submission."

#### Scenario: DV analytics respect small-cell suppression
- **WHEN** an admin asks about DV shelter utilization
- **THEN** the agent calls `get_dv_summary`
- **AND** the response applies small-cell suppression (D18): counts suppressed when < 3 shelters or < 5 beds
- **AND** the agent notes: "Some figures are suppressed to protect DV shelter identity per small-cell suppression rules."
