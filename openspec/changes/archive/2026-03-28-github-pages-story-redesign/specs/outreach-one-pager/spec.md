## outreach-one-pager

Printable one-page HTML document for first outreach conversations.

### Requirements

- REQ-PAGER-1: The one-pager MUST be a single printable HTML page (A4/Letter) with `@media print` styles
- REQ-PAGER-2: The one-pager MUST lead with the problem story, not feature list
- REQ-PAGER-3: The one-pager MUST include separate sections for outreach workers (what they can do) and coordinators (what changes for them)
- REQ-PAGER-4: The one-pager MUST include what it costs (free software, hosting costs, who typically pays)
- REQ-PAGER-5: The one-pager MUST be honest about what's not done yet
- REQ-PAGER-6: The one-pager MUST fit on one printed page — no scrolling required when printed
